## twitterExpand

## scryfall
access the scryfall API in chat 

| Format | Function |
| ------ | -------- |
| [[growth spir]] | fuzzy search |
| [[!growth spiral]] | exact search |
| [[?]] | random card |
| [[?type:instant cmc:2]] | random card with scryfall filters |
